# NLP 100本ノック 2025

## Environment Set Up

1. Install dependency by uv

```bash
uv sync
```

2. Activate the virtual environment and run script

```bash
. .venv/bin/activate
 (nlp100) > python3 <python script>
```

or runnign python script without virtual environment activation

```bash
uv run <python script>
```
